Project Instructions

1) Add the files to a directory of your choice
2) Load "smalldblib" into Netbeans 
3) Right Click "smalldblib" and Select "Build" to compile
3) Load "smallAsg1Client" into Netbeans
4) Right Click and Select "Properties"
5) Click "Libraries" on the left-hand side
6) Click "Add JAR/FOLDER" and add the JDBD .jar file stored under your driver's directory
7) Click "Add Project" and add the smalldblib project located under the directory you originally picked
8) Right Click "smallAsg1Client" and Select "Build" to compile
9) Right Click "smallAsg1Client" and Select "Run" to run the project
