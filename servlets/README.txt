Installation Instructions
_________________________

1) Load "smalldblib" into NetBeans
2) Build and Deploy
3) Load "small1_beychok2asg3" into NetBeans
4) Add "smalldblib.jar" from "smalldblib/dst"
5) Start Glassfish
6) Add the Level 4 Driver to Glassfish/lib if you haven't done so already
7) Build and Deploy "small_beychok2asg3"
